## A Modern Calculator: A Web Development Project

**Project Overview**

This project is a sleek and functional calculator built with HTML, CSS, and JavaScript. It provides a user-friendly interface for performing basic arithmetic operations and includes additional features like trigonometric functions, logarithms, and percentages.

**Features**

* **Intuitive interface:** Modern design with clear button labels and a responsive layout.
* **Comprehensive functionality:** Supports addition, subtraction, multiplication, division, trigonometric functions (sin, cos, tan), logarithms (log, ln), percentages, and square root.
* **Error handling:** Prevents invalid inputs and displays informative error messages.
* **Keyboard support:** Allows users to input numbers and operators using their keyboard.

**Technologies Used**

* **HTML:** Structure and layout of the calculator.
* **CSS:** Styling and appearance of the calculator.
* **JavaScript:** Handles user input, performs calculations, and updates the display.

**Installation**

1. **Clone the repository:**
   ```bash
    https://github.com/sarnagithub/Calculator
   ```
2. **Open the project:**
   Navigate to the project directory and open the `MYcalculator.html` file in a web browser.
